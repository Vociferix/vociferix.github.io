Expand description

Input derived from other input by repeatedly applying a parser.

ParsedInput consists of an underlying input and a parser. Each symbol produced by a ParsedInput is produced by applying the parser to the inner input. After parsing a symbol, the inner input becomes the remaining unparsed inner input. The ParsedInput ends when parsing a symbol fails.
